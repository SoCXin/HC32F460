平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示SPI模块4线式主从机中断传输数据功能。

说明：
本样例配置SPI为4线式，通过宏开关选择主机或从机模式，将主机代码下载到一块板子，从
机代码下载到另一块板子，通过排线将两块板子的SPI接口相连接，主机按下K1按键，触发
一次数据收发，主机和从机对收到的数据和发送的数据进行比较，相同的LED_GREEN亮，不
同则LED_RED亮。

使用步骤
================================================================================
1）打开工程，启动IDE的下载和调试功能，分别为主机和从机下载程序；
2）先重新编译工程后下载到主机；
3）再注释掉“SPI_MASTER_MODE”宏定义，并打开“SPI_SLAVE_MODE”宏定义，
重新编译后下载到从机中；
4）按下主机的K1按键触发主机、从机传输数据，并在传输完成后执行比较功能；
5）比较结果相同则LED_GREEN亮，不同则LED_RED亮；
6）可重复按下K1后执行4、5的操作。

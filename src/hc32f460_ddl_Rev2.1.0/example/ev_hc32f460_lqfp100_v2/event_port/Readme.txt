平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为Event Port样例。
样例中，使用EventPort2.1触发EventPort2事件，再由EventPort2触发EventPort4事件，并
从EventPort4.3输出驱动LED_RED点亮；同时也提供了EventPort4被触发后的端口
（EventPort4.15）状态读取示例。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）按下K10来产生EventPort2事件；
4）观察由EventPort4.3所控制的LED_RED变化。

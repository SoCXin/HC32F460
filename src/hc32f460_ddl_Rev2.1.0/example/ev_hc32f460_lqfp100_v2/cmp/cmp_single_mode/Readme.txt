平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示CMP的单一比较功能。

无

使用步骤
================================================================================
1）CMP2_INP1(PA4)与ADC电位器输出(PC0)连接，VCOUT2(PB13)与示波器连接；
2）打开工程编译并全速运行；
3）调节ADC电位器；
4）观察LED_GREEN与示波器：
   a、PA4输入为高电平时，LED_GREEN点亮，VCOUT2输出高电平，表示INP电压高于INM；
   b、PA4输入为低电平时，LED_GREEN熄灭，VCOUT2输出低电平，表示INP电压低于INM；

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示QSPI四线式输入输出快速读功能。

说明：
本样例设置QSPI四线式输入输出快速读模式，每按一次K1则触发一次擦除flash、写入数据
到flash、从flash读取数据，再讲读写的数据进行比较，相同的LED_GREEN亮，不同则
LED_RED亮，每次按键都将flash地址偏移到下一个扇区。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）按下K1触发flash擦除、写入、读出后比较功能；
4）比较结果相同则LED_GREEN亮，不同则LED_RED亮；
5）重复按下K1改变flash读写地址，再执行3、4的操作。

功能描述
================================================================================
本样例使用I2S外设全双工模式，通过WM8731同步录取和播放音频。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-F460-LQ100-V2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）全速运行后，短按按键K1，LED_RED亮，启动全双工的录音和播放，
   通过MIC进行录音，通过J15耳机接口播放；
4) 短按K2，LED_RED灭，全双工录音和播放停止。

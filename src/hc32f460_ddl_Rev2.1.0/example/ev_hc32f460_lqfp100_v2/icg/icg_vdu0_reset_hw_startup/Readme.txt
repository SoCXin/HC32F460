平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示ICG模块中VDU0功能。

说明：
关于ICG相关配置需参照mian()函数中的描述去修改hc32f460_icg.h中的宏定义。

本样例设置硬件启动VDU0且设置电压阈值为2.3V，调整供电电压低于阈值进入复位状态，然
后升高电压到3.3V使芯片进入正常工作状态。

使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）常规复位后，LED_RED闪烁；
4）将目标板供电电压调至2.2V，此时芯片进入复位状态，LED_RED不在闪烁；
5）将电压调至3.3V，芯片退出复位状态，进入正常工作状态，但检测到是欠压复位，此时
LED_RED常亮。

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为时钟频率测量的数字滤波功能，外部管脚输入参考时钟32.768KHZ，当被测时钟超过
4M，发生频率异常中断，滤波时钟为0.5M。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-F460-LQ100-V2.0

辅助工具:
---------------------
信号发生器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）连接信号发生器与PH2，调整信号发生器频率为32.768KHZ以下；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行；
4）可观察到绿灯亮；
5）更改被测时钟分频（使其大于4M），重复步骤1）~3），可观察到红灯亮。


================================================================================
注意
================================================================================
也可使用switch_sys_clk样例，调节output时钟频率2M以下，作为信号发生端。

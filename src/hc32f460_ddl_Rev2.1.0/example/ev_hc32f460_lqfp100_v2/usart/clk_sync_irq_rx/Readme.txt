平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示USART外设配置为时钟同步模式时通过中断方式收发数据。

使用步骤
================================================================================
1）需两个测试板，按照如下方式连接。
   测试板A:J5    测试板B:J5
      TX-------------RX
      SCK-----------SCK
      GND-----------GND
2）打开工程clk_sync_irq_tx并重新编译，下载代码测试板A。
   打开工程clk_sync_irq_rx并重新编译，下载代码测试板B。
3）先按下测试板A复位键，再按下测试板B复位键。
4）按下测试板A按键K1，测试板A LED_GREEN点亮，表示数据发送完成。
5）测试板B LED_GREEN点亮，表示测试板B接收的数据符合预期。

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
1）例程实现了 ADC1 PGA（只有 ADC1 有 PGA 功能）的配置和用法;
2）用串口（波特率 115200）调试助手可查看 PGA 放大前后的数据对比。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-F460-LQ100-V2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手（115200bps，8N1）

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）Micro USB 插入 USBDAP 将开发板与电脑相连；
3）将电位器输出端与引脚 ADC1_IN0(PA0) 相连，使电位器输出到 ADC1_IN0；
4）启动 IDE 的调试功能，断点查看程序运行情况；或全速运行用串口调试助手查看 PGA 放大
   前后的数据对比。
5）旋转电位器，可改变 PGA 输入；

================================================================================
注意
================================================================================
1）只有 ADC1 有 PGA 功能；
2）PGA 通道直接对应外部引脚，ADC1 序列 A 或 序列 B 选择的通道对应的引脚，必须与 PGA
   通道对应的引脚一致；
3）PGA 输入电压必须满足：
   最大值：0.9*Vcca/Gain
   其中：Vcca：模拟电源电压，最小值 1.8V，典型值 3.3V，最大值 3.6V；
         Gain：放大倍数。

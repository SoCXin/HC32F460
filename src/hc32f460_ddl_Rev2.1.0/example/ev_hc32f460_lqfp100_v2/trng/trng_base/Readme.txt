平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
1）实现了真随机数发生器（TRNG）的配置和基本用法；
2）用延时的方式每 100 毫秒启动一次 TRNG 以产生随机数，并通过串口（波特率 115200）
   将随机数打印出来。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-F460-LQ100-V2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
串口助手软件(115200bps, 8N1)

================================================================================
使用步骤
================================================================================
1）用 IDE 打开工程并重新编译；
2）Micro USB 数据线插入 USBDBG 将开发板与电脑相连；
3）启动 IDE 调试功能，断点查看随机数值，或全速运行程序，用串口调试助手查看随机数。

================================================================================
注意
================================================================================
1）请将随机数时钟频率设置在 1MHz（含）以内，如果不能，请将其时钟频率尽可能设低。

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示Timera模块捕获及外部中断事件触发捕获功能。

说明：
本样例设置Timera为锯齿波、增计数模式，Timera单元1的通道1、2开启捕获上升沿输入功
能及捕获中断，同时使能K10触发通道1、2捕获功能，通道1捕获PA8的输入则切换LED_RED状
态，通道2捕获PE11的输入则切换LED_GREEN状态。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）通过跳线将PA8和PE11引脚交替拉高，观察LED_RED和LED_GREEN状态；
4）PA8的输入则切换LED_RED状态，PE11的输入则切换LED_GREEN状态，则功能正常；
5）按下K10按键则触发通道1、2捕获，同时改变LED_RED、LED_GREEN状态。

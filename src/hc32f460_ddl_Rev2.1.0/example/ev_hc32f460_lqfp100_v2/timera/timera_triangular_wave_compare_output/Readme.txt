平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示Timera模块PWM输出及外部中断触发Timer启动功能。

说明：
本样例设置Timera为三角波计数模式，Timera单元1的通道1、3配置为PWM输出模式，通过按
键K10启动Timera输出，启动成功后LED_RED会闪烁，通过按键K2改变输出占空比，可以将示波
器连接到PA8（通道1）、PE13（通道3）观察输出波形。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）按键K10启动Timera的PWM输出功能，观察LED_RED闪烁；
4）将示波器连接到PA8、PE13观察输出波形，反复触发按键K2改变输出占空比；
5）观察示波器波形变化。

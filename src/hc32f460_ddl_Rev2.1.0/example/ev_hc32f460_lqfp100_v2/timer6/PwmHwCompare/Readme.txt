平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER6的带硬件死区的PWM输出功能，
说明：
配置timer6，hclk设置为200M,死区时间16us
================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-F460-LQ100-V2.0

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  打开工程编译并全速运行。
2） 使用示波器测试PA08和PE08的输出波形，
    看到PA08(Timer61 PWMA)和PE08（Timer61 PWMB）输出PWM波形，PWMA的占空比一大一小交替，
    PWMB与PWMA反相，
    并且PWMB的下降沿早于PWMA的上升沿16us，
        PWMB的上升沿晚于PWMA的下降沿16us。

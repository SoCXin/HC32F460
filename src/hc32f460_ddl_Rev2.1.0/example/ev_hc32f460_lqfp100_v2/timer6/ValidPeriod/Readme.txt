平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER6的PWM输出功能(使能有效周期，每隔1个周期产生一次中断和触发事件）
说明：
配置timer6，hclk设置为168M，死区时间16us

Timer6通过AOS功能触发ADC1启动转换，触发事件为Timer6的下溢事件，或者SCMA的向上计数比较匹配事件（INTENSAU）（当SCMA_ValidPeriod define有效时候）。

使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） 使用示波器测试PA08和PE08的输出波形，
    看到PA08(Timer61 PWMA)和PE08（Timer61 PWMB）输出PWM波形，PWMA的占空比大小交替，
    两个周期变换一次
    PWMB与PWMA反相，
    并且PWMB的下降沿早于PWMA的上升沿16us，
        PWMB的上升沿晚于PWMA的下降沿16us。
3)  示波器查看LED_GREEN在ADC1中断内翻转电平，可看到LED_GREEN的电平每隔一个PWM周期翻转一次

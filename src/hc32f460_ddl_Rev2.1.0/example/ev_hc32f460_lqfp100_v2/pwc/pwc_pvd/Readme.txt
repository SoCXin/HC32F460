平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为PVD1可屏蔽中断，检测电压2.8V，PVD2不可屏蔽中断检测电压2.3V。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-F460-LQ100-V2.0

辅助工具:
---------------------
电压源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）断开J20跳冒，使用电压源给芯片VCC引脚供电，并设置电压为3.3V；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行；
4）缓慢调节电压源电压至2.8V以下（2.7V）,可观察到LED_RED灯亮；
5）继续调节电压源电压至2.3V以下（2.2V），可观察到LED_RED灯灭，LDE_YELLOW灯亮

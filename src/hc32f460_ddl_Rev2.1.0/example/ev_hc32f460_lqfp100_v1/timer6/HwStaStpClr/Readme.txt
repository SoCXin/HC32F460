平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER6的硬件启动、停止、清零功能
说明：
配置timer6，hclk设置为200M
PE6-->PWMA
PD7-->PWMB

硬件启动: PE6-->PWMA 上升沿

硬件停止: PE6-->PWMA 下降沿

硬件清零: PD7-->PWMB 上升沿

================================================================================
使用步骤
================================================================================
1） PE6连接PA8，PD7连接PE8
2） 打开工程编译并全速运行。
3） 使用示波器观察PE6、PD7、PB5端口
PE6上升沿时候，PB5有翻转电平输出
PE6下降沿的时候，PB5停止翻转，此时Timer61的CNTER停止计数
PD7上升沿的时候，Timer61的CNTER被清零

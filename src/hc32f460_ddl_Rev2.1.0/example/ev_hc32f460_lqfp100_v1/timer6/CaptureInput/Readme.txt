平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER62的PWMA端口的捕获输入功能(无缓存功能）
说明：
配置timer6，hclk设置为200M，PLCK0为200M
Timer61和Timer62的计数时钟为PCLK0/1024

================================================================================
使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） Timer61的PWMA引脚（PA08）交替输出3种占空比的PWM波形（比较值分别为0x2000,0x4000,0x6000，周期为0x8000）
3） 用引线连接Timer62的PWMA引脚（PA09）
4)  在live watch窗口可看到u16CaptureA的值交替变化，
且值与timer61的比较值一致

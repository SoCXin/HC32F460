平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER0内部硬件触发功能。
说明：
TIMER0 UNIT2 CHANEL B配置为内部时钟计数的输入捕获模式，触发方式配置为触发有效，启动计数。
使用SW2产生的Exint事件作为TIMER0 外部触发输入

================================================================================
使用步骤
================================================================================
1）  打开工程编译并全速运行，
2）  按下按键SW2触发TIMER0捕获中断EIRQ1，LED_REGGN亮，按下按键再次触发TIMER0捕获
     中断EIRQ1，LED_REGGN灭。
3）  当采集到一次宽度数据后，LED_BLUE常亮

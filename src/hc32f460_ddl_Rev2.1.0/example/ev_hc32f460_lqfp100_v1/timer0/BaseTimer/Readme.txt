平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER0的基本计数功能，在中断中翻转led0和led1。
说明：
使能TIMER0 UNIT2 CHANEL A计数匹配中断，异步计数方式，XTAL32作为计数时钟源频率32K,
中断频率约为0.5S一次;
使能TIMER0 UNIT2 CHANEL B计数匹配中断，同步计数方式，PCLK1作为计数时钟源频率100M,
中断频率约为0.5S一次;

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示USART外设配置为UART模式时收数据通过硬件流控RTS。

无

使用步骤
================================================================================
1）需两个测试板，按照如下方式连接。
   测试板A:J20    测试板B:J20
      RX-------------TX
      RTS-----------CTS
      GND-----------GND
2）打开工程uart_hw_flow_ctrl_rx_rts并重新编译，下载代码测试板A。
   打开工程uart_hw_flow_ctrl_tx_cts并重新编译，下载代码测试板B。
3）按下测试板B复位键，再按下测试板A复位键。
4）按下测试板B按键SW2。
5）等待几秒，测试板A LED_GREEN点亮，表示测试板A数据发送完成。
6）测试板B LED_GREEN点亮，表示测试板B接收的数据符合预期。

注意
================================================================================
如使用时更改USART通道，需同时修改UART外设通道，GPIO配置、外设Clock使能和配置
NVIC功能的相应事件号。

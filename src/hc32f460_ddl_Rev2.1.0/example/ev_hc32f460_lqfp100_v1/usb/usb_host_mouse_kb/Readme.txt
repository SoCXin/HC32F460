平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU USB外设实现的的USB host HID mouse & keyboard功能。
说明：

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.3

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手

================================================================================
使用步骤
================================================================================
1）  打开工程编译并全速运行，
2）  连接OTG转接线，再接上鼠标或键盘，
3）  打开串口调试助手，设置波特率为115200bps，
4）  移动鼠标或键盘按键，串口调试助手界面打印相应鼠标坐标或键盘键值信息。

================================================================================
注意
================================================================================
使用开发板MicroUSB接口J14
J1串口打印波特率115200

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的USB外设实现的键盘功能的HID应用。
评估板上的按键对应的键值如下：
按下SW4对应键值'A'
按下SW5对应键值'B'
按下SW3对应键值'C'
按下SW2对应键值'D'

================================================================================
使用步骤
================================================================================
1）  打开工程编译并全速运行，
2）  使用USB连接线接到PC，
3）  打开一个text文本，定位好光标，点击按键SW4/SW5/SW3/SW2，将从text文本中输出A/B/C/D

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为I2C外设从机模式进行中断方式的数据收发样例

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）准备另一个测试板下载样例i2c_master_irq。
3）连线：主机SDA->从机SDA；主机SCL->从机SCL；主机GND->从机GND。
4）启动从机板子IDE的下载和调试功能，全速运行；
5）全速运行主机板子的i2c_master_irq样例。
6）从机进行完数据的接收和发送后，红灯闪烁表示通信错误。
7）主机接受到从机返回的正确数据后，绿灯闪烁，如主机通信失败，红灯闪烁。

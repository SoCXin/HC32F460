平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示如何使用Timer4脉冲宽度调制模块PWM。

说明：
样例配置比较输出模块OCO偶通道，使能OCCR缓冲功能，并通过PWM直通模式，输出其波形。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.3

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板PE9(PWM_H)引脚与示波器相连；
2）编译、下载并运行；
3）观察示波器，PE9通道波形如下，高电平为5:2:5, 低电平：5:5:2。
      _____       __       _____    _____       __       _____
    _|     |_____|  |_____|     |__|     |_____|  |_____|     |__|

================================================================================
注意
================================================================================
若更换Timer4单元或Timer4 PWM通道，请连接对应的PWM引脚。

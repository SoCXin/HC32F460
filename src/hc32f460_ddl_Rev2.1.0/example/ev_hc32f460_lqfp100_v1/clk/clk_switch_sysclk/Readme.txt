平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为系统时钟切换，按下不同的按键，系统时钟切换为不同的频率,示波器所测量时钟为系统
时钟的八分频。
SW6  -->  MRC     -->  8MHz       --> 1Mhz
SW7  -->  XTAL    -->  8MHz       --> 1MHz
SW8  -->  HRC     -->  16MHz      --> 2MHz
SW9  -->  LRC     -->  32.768KHz  --> 4096Hz
SW10 -->  XTAL32  -->  32.768KHz  --> 4096Hz
SW11 -->  MPLL    -->  200MHz     --> 25MHz

================================================================================
使用步骤
================================================================================
1）连接PA8与示波器
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行，停在while（1）；
4）查看变量stcClkFreq，各时钟频率与预期是否相符。
5）查看示波器，系统时钟频率与预期是否相符。

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示ICG模块中SWDT中断功能。

说明：
关于ICG相关配置需参照mian()函数中的描述去修改hc32f460_icg.h中的宏定义。

本样例设置硬件启动SWDT，并设置约1.6秒触发一次SWDT计数溢出中断，通过K2（SW2）循环
切换工作模式（正常模式、睡眠模式、停止模式），正常模式LED_RED闪烁，睡眠模式
LED_GREEN闪烁，停止模式LED_RED和LED_GREEN同时闪烁。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，关闭IDE下载界面；
3）复位按键复位芯片，进入正常模式LED_RED闪烁；
4）按下K2（SW2）切换工作状态到睡眠模式，LED_GREEN闪烁；
5）再次按下K2（SW2）切换工作状态到停止模式，LED_RED和LED_GREEN闪烁；
6）再次按下K2（SW2）切换工作状态到正常模式，LED_RED闪烁；

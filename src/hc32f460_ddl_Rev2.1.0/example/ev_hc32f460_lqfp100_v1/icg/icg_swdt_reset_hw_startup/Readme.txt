平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示ICG模块中SWDT复位功能。

说明：
关于ICG相关配置需参照mian()函数中的描述去修改hc32f460_icg.h中的宏定义。

本样例设置硬件启动SWDT，并以约1.2秒进行一次喂狗操作，正常启动情况下LED_RED闪烁；按
下K2（SW2）则触发错误区间刷新动作，导致SWDT触发复位动作，SWDT触发复位后LED_RED常亮。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）复位按键复位芯片，LED_RED闪烁；
3）按下K2（SW2）触发错误刷新动作；
4）SWDT触发复位后LED_RED常亮。

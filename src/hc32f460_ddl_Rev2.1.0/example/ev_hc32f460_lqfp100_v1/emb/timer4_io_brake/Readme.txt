平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的EMB模块的GPIO输入刹车功能。
说明：
EMB2用于Timer4，刹车端口为PB12/PE15/PA06
（EMB2-4用于Timer4）
EMB2中断内设置软件刹车，清除PortIn刹车标志
在while（1）内做刹车处理，处理完后使能PWM输出

================================================================================
使用步骤
================================================================================
1）示波器连接PE9，PB12连接VCC
2）编译，下载程序并全速运行
3）PE9输出PWM波形，LED_RED熄灭
4）PB12连接GND，PE9输出变为低电平，LED_RED点亮。
5）PB12再次连接VCC，约3秒后，PE9恢复输出PWM波形，LED_RED熄灭。
6）重复步骤4）~步骤5）

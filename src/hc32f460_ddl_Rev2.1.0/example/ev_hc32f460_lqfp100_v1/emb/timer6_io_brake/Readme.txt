平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的EMB模块的GPIO输入刹车功能。
说明：
EMB1用于Timer6，刹车端口为PA11或PB02
（EMB2-4用于Timer4）
 EMB1中断内设置软件刹车，清除PortIn刹车标志
在while（1）内做刹车处理，处理完后使能PWM输出

================================================================================
使用步骤
================================================================================
1） 示波器查看PE9和PE8，PA11连接VCC
2） 编译，下载并全速运行。
3） PE9和PE8输出PWM波形，LED_RED熄灭
4） PA11连接低电平，可以看到PWM输出变为Low level，LED_RED亮起
5） PA11重新接Vcc，约3秒后，PE9和PE8恢复输出PWM波形，LED_RED熄灭。
6） 重复步骤4）~步骤5）

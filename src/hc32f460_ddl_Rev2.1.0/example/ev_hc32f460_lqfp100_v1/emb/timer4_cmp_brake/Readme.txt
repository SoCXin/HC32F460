平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的EMB模块的比较刹车功能。
说明：
EMB2用于Timer4，CMP1 INP使用PA00，INM使用PC03或者DAC1
（EMB2-4用于Timer4）
 EMB2中断内设置软件刹车，清除CMP刹车标志
在while（1）内做刹车处理，处理完后关闭软件刹车，使能PWM输出
================================================================================

================================================================================
使用步骤
================================================================================
1） 示波器连接PE9，断开跳冒J24，连接电位器输出到PA0，调节电位器输出为最小
2） 编译，下载程序并全速运行
3） 示波器可以看到PWM输出，调节电位器输出大于CMP1 INM端电压，PWM输出关闭，LED_RED点亮
4） 重新调节电位器小于CMP1 INM端电压，延时约3秒之后PWM恢复输出，LED_RED熄灭。

================================================================================
注意
================================================================================
CMP输出高的时候，EMB刹车有效，当清除CMP刹车标志并退出中断，PWM会立刻恢复输出，
例程在CMP中断内设置软件刹车使能，然后主循环内停机处理完成后关闭软件刹车

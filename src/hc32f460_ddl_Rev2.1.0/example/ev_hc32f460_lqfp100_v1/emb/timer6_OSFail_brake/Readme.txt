平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的EMB模块的XTAL故障刹车功能。
说明：
EMB1用于Timer6，
（EMB2-4用于Timer4）
 EMB1中断内设置软件刹车，执行软复位

无

================================================================================
使用步骤
================================================================================
1） 示波器连接PE9和PE8
2） 编译，下载并全速运行。
3） 示波器观察到PWM输出，用镊子短接XTAL引脚，人为造成XTAL故障
4） PWM输出停止，LED_RED点亮
5） 按下复位键产生复位，示波器观察到PWM输出，LED_RED熄灭
6） 重复步骤4）~ 5）

平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例使用I2C外设通过WM8731录取和播放音频。

================================================================================
使用步骤
================================================================================
1）打开工程
2）根据硬件情况配置，并重新编译；
   如对MIC进行录音则打开宏定义      #define RECORD_MIC      true
   如对LINEIN进行录音则关闭宏定义   #define RECORD_MIC      false
3）启动IDE的下载和调试功能，全速运行
4）全速运行后，短按按键K2，LED_RED亮，录音开始，红灯灭录音结束
5) 录音结束后，短按按键K2，循环播放录音。
